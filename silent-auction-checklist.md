# Silent Auction Planning Checklist

## 8 Weeks Before

- Set fundraising goal
- Choose event date
- Select venue
- Build organizing committee

## 6 Weeks Before

- Request donated auction items
- Contact sponsors
- Promote the event

## 4 Weeks Before

- Photograph auction items
- Write descriptions
- Publish auction catalog

## 2 Weeks Before

- Recruit volunteers
- Test bidding platform
- Print event materials

## Event Day

- Set up auction tables
- Register guests
- Monitor bidding
- Process payments

## After Event

- Thank donors
- Thank bidders
- Share fundraising results
